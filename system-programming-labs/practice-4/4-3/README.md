# Задание 4.3 — Демоны и фоновые процессы

## Условие
**Вариант 2**: Демон создаёт файл `pidfile.txt` со своим PID и удаляет его при завершении.

## Требования
- Использовать `fork()`, `setsid()`, `chdir()`, `umask()`, `close()`
- Организовать запись событий в лог-файл
- Обрабатывать сигналы завершения (SIGTERM, SIGINT)
- Компиляция: `gcc -Wall -Wextra -o task task.c`

## Компиляция и запуск
```bash
gcc -Wall -Wextra -o task task.c
./task
