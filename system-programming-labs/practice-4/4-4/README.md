# Задание 4.4 — Анализ памяти и процессов

## Условие
**Вариант 2**: Использовать `cat /proc/[PID]/maps` для анализа сегментов процесса.

## Требования
- Использовать `ps`, `pmap`, `top`, `/proc/[PID]/maps`, `/proc/[PID]/status`, `valgrind` или `strace`
- Сохранить команды и результаты в `analysis.txt`
- В `README.md` описать выводы

## Используемые утилиты

| Утилита | Назначение | Пример |
|---------|------------|--------|
| `ps` | просмотр процессов | `ps aux` |
| `top` | динамический мониторинг | `top -p PID` |
| `pmap` | карта памяти процесса | `pmap -x PID` |
| `/proc/PID/maps` | сегменты памяти | `cat /proc/PID/maps` |
| `/proc/PID/status` | статус процесса | `cat /proc/PID/status` |
| `valgrind` | проверка утечек памяти | `valgrind --leak-check=full ./program` |
| `strace` | трассировка системных вызовов | `strace -c ./program` |
