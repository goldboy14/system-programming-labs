# Практика 2: Потоки, синхронизация, сигналы

Содержит 4 задания по продвинутым темам системного программирования:

1. **2-1:** Многопоточность с использованием библиотеки pthread
2. **2-2:** Синхронизация потоков с помощью мьютексов
3. **2-3:** Обработка сигналов с использованием signal()/sigaction()
4. **2-4:** Отладка программ с помощью strace

Все задания соответствуют варианту 2 из методички.
