# Задание 2-1: Многопоточность

## Вариант 2
Создать 2 потока. Первый считает сумму 1-1000, второй — произведение 1-10. 
Потоки возвращают результат через `pthread_exit()`. 
Главный поток выводит оба результата.

## Требования
- Использование библиотеки pthread
- Функции: `pthread_create()`, `pthread_join()`, `pthread_exit()`
- Компиляция с флагом `-pthread`

## Описание программы
Программа демонстрирует создание и управление потоками:
1. **Главный поток** создает два рабочих потока
2. **Первый поток** вычисляет сумму чисел от 1 до 1000
3. **Второй поток** вычисляет произведение чисел от 1 до 10
4. Рабочие потоки возвращают результаты через `pthread_exit()`
5. Главный поток получает результаты через `pthread_join()` и выводит их

```bash
# Компиляция с поддержкой потоков
gcc -pthread -Wall -Wextra -o task task.c

# Запуск программы
./task
