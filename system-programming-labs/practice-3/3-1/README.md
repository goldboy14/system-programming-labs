# Задание 3.1 — Неименованные каналы (pipe)

## Условие
Вариант 2: Родитель передаёт дочернему массиву из 10 целых чисел. Дочерний вычисляет сумму и выводит результат.

## Требования
- Использовать `pipe()`, `fork()`, `read()`, `write()`, `close()`
- Не использовать высокоуровневый ввод-вывод (`fopen()`, `fprintf()`, `fscanf()`)
- Обрабатывать ошибки через `perror()`
- Компиляция: `gcc -Wall -Wextra -o task task.c`

## Ход выполнения
1. Родитель создаёт неименованный канал с помощью `pipe()`
2. Родитель заполняет массив из 10 элементов случайными числами
3. Родитель создаёт дочерний процесс через `fork()`
4. Родитель закрывает ненужный конец канала (для чтения)
5. Родитель передаёт массив через канал с помощью `write()`
6. Дочерний процесс закрывает ненужный конец канала (для записи)
7. Дочерний читает массив через `read()` и вычисляет сумму
8. Дочерний выводит результат
9. Родитель ожидает завершения дочернего процесса через `wait()`

## Компиляция и запуск

### Компиляция
```bash
gcc -Wall -Wextra -o task task.c
