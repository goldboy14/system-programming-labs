# Задание 3.4 — Статистика сетевых соединений

## Условие
Использовать утилиты `ss`, `netstat`, `lsof`, `ping`, `curl`, `nc` для анализа сетевых соединений.

## Требования
- Сохранить команды и результаты в `commands.txt`
- Кратко описать наблюдения в `README.md`
- Использовать `ping` для проверки доступности localhost

## Используемые утилиты

| Утилита | Назначение | Ключевые опции |
|---------|------------|----------------|
| `ping` | Проверка доступности хоста | `-c` (количество пакетов) |
| `ss` | Просмотр статистики сокетов | `-t` (TCP), `-u` (UDP), `-l` (LISTEN), `-n` (числовые порты) |
| `netstat` | Просмотр сетевых соединений | `-t`, `-u`, `-l`, `-n`, `-a` |
| `lsof` | Список открытых файлов | `-i :порт` (поиск по порту) |
| `curl` | HTTP-запросы | `-I` (заголовки), `-s` (тихий режим) |
| `nc` | Работа с TCP/UDP | `-z` (сканирование), `-v` (подробно), `-l` (слушать) |
