# Задание 1-1: Системные вызовы

## Вариант 2
Создать файл text.txt с текстом (200+ символов), подсчитать буквы 'a' и 'A', 
записать результат в count.txt.

## Требования
- Используются только системные вызовы: open(), read(), write(), close()
- НЕ используются: fopen(), fprintf(), fscanf()
- Обработка ошибок с помощью perror()

## Описание программы
Программа выполняет следующие действия:
1. Создает файл `text.txt` с текстом объемом более 200 символов
2. Открывает файл для чтения с помощью `open()`
3. Читает файл по частям с помощью `read()`
4. Подсчитывает количество букв 'a' и 'A'
5. Записывает результат в файл `count.txt` с помощью `write()`

# Компиляция

# Запуск
