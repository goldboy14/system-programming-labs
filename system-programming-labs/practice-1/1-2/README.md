# Задание 1-2: Процессы fork/exec

## Вариант 2
Создать 2 процесса. Дочерний запускает `execl("/bin/ls", "ls", "-la", NULL)`. 
Родитель ждёт и выводит "Команда ls завершена".

## Требования
- Используются: fork(), exec(), wait(), getpid(), getppid()
- Обработка ошибок для всех системных вызовов

## Описание программы
Программа демонстрирует:
1. Создание дочернего процесса с помощью `fork()`
2. Замену образа дочернего процесса на команду `ls -la` с помощью `execl()`
3. Ожидание родительским процессом завершения дочернего с помощью `waitpid()`
4. Вывод информации о PID и PPID обоих процессов

# Компиляция
gcc -Wall -Wextra -o task task.c

# Запуск
./task
