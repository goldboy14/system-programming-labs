# Задание 1-3: Файловая система и права

## Вариант 2
Создать вложенные директории `level1/level2/level3`. 
В `level3` создать `secret.txt` с правами `600`. 
Вывести полный путь.

## Требования
- Bash скрипт
- Работа с правами доступа (chmod)
- Вывод полного пути к файлу

## Описание скрипта
Скрипт `script.sh` выполняет:
1. Создание вложенной структуры директорий `level1/level2/level3`
2. Создание файла `secret.txt` с конфиденциальной информацией
3. Установку прав доступа `600` (только владелец может читать и писать)
4. Проверку установленных прав
5. Вывод полного абсолютного пути к файлу
6. Демонстрацию работы системы прав доступа

## Запуск скрипта
```bash
# Даем права на выполнение
chmod +x script.sh

# Запускаем скрипт
./script.sh
