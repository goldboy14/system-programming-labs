# Практика 1: Системные вызовы, процессы, файлы

Содержит 4 задания по основам системного программирования:

1. **1-1:** Работа с файлами через системные вызовы
2. **1-2:** Создание процессов с fork() и exec()
3. **1-3:** Управление файловой системой через bash скрипты
4. **1-4:** Диагностика системных процессов

Все задания соответствуют варианту 2 из методички.
